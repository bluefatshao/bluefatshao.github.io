男朋友教程第一弹

typescript

  1. 简介：想必男朋友肯定知道JavaScript，我就不赘述了，TypeScript是JavaScript的一个超集，也就是说现有的JavaScript代码可以丝滑地与TypeScript一起运行，无语任何修改，TypeScript 可处理已有的 JavaScript 代码，并只对其中的 TypeScript 代码进行编译。

  2. TypeScript安装

    安装需要用到npm工具，打开终端，输入以下命令，就可以安装成功了。

    npm install -g typescript

    然后再检查一下是否安装成功，需要用到tsc，在终端下面输入以下命令，会显示你的TypeScript的版本。

    tsc -v

  3. 简单地尝试

    然后我们新建一个app.ts的文件，在文件中输入如下代码：*(通常来说.ts是TypeScript代码文件的扩展名。)*

    var message:string = "毛毛是个小可爱"

    console.log(message)

    然后保存，在终端执行以下命令将TypeScript转换为JavaScript代码：

    tsc app.ts

    你会发现，在当前目录下（与app.ts同一目录）会生成一个app.js文件，使用node命令来执行app.js文件：

    node app.js

    然后就会输出毛毛是个小可爱了。

耶！你现在已经掌握了什么是TypeScript，如何安装TypeScript，如何运行第一个代码了！
